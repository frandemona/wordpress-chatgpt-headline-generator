This plugin uses OpenAI's ChatGPT API to generate a list of catchy headlines based on the keywords in a post's description.

== Description ==

= Requirements =
- A WordPress website
- An API key from OpenAI's ChatGPT API

= Installation =
1. Download the plugin from the repository and upload it to your WordPress site.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the "ChatGPT Headline Generator" settings page under the "Settings" menu and enter your API key.

= Usage =
1. Create a new post in WordPress.
2. In the post editor, write a description of the post.
3. Publish the post.
4. The plugin will automatically generate a list of catchy headlines based on the keywords in the post description and display them on the post's edit screen.
5. Once a title is added to the post and saved the plugin won't trigger the headline generator. To trigger again you can save the post again with an empty title or the title "generate title"
